## Программа для просмотра текстовых файлов

#### Requirements
- `sudo apt-get install libncurses5w-dev`

#### Quick start
- `make` / `make build` - собрать исполняемый файл `Show`
- `./Show file.txt`- открыть файл для просмотра
- `make clean` - удалить исполняемые файлы
